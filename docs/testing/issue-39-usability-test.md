# Issue #39 usability test

Use this protocol with 2–3 physics students. Do not coach participants after
the task begins. Record results anonymously and obtain consent before taking
notes or screen recordings.

## Task

> Upload the supplied projectile video, select the useful interval, calibrate
> it, run the YOLO analysis, inspect the trajectory, and export the result.

Start timing when the participant begins interacting with ArcLab. Stop timing
when a CSV or PDF has been exported.

## Acceptance targets

- Task completion without help: at least 80%
- Median time on task: under 2 minutes
- Mean System Usability Scale (SUS): at least 68

## SUS questionnaire

Ask participants to rate every statement from 1 (strongly disagree) to
5 (strongly agree).

1. I think that I would like to use ArcLab frequently.
2. I found ArcLab unnecessarily complex.
3. I thought ArcLab was easy to use.
4. I think that I would need technical support to use ArcLab.
5. I found the functions in ArcLab well integrated.
6. I thought there was too much inconsistency in ArcLab.
7. I imagine that most people would learn to use ArcLab quickly.
8. I found ArcLab cumbersome to use.
9. I felt confident using ArcLab.
10. I needed to learn a lot before I could use ArcLab.

For odd-numbered questions, subtract 1 from the response. For even-numbered
questions, subtract the response from 5. Add the adjusted values and multiply
the total by 2.5 to obtain a score from 0 to 100.
